聚类模型案例：癌细胞基因表达数据

吴翔

2019-03-20

概述

我们通过R语言ISLR包中64个细胞的6830个基因表达数据作为案例，来阐述如何使用如下聚类模型：

  • 系统聚类
  • \(K\)均值聚类

数据集NCI60包含基因表达数据矩阵NCI60$data以及癌细胞系的类型NCI60$labs。

# clean the work directory
rm(list = ls())

# set seeds
set.seed(123)

# read dataset
suppressMessages(library(ISLR))
suppressMessages(library(tidyverse))
data("NCI60")

系统聚类

# scaling
ncidat <- scale(ncidat)

# clustering
nci.dist <- dist(ncidat, method = "euclidean", p = 2)
hc.out <- hclust(nci.dist)
plot(hc.out, labels = ncilab, main = "complete linkage", xlab = "", ylab = "")

在以上谱系图上某个高度切割，可以得到指定类数的聚类，例如类别数量\(s = 4\)。

# four clusters
table(cutree(hc.out, 4))
## 
##  1  2  3  4 
## 40  7  8  9
# plot
plot(hc.out, labels = ncilab)
abline(h = 139, col = "red")

\(K\)均值聚类

# kmeans
km.out <- kmeans(ncidat, 4, nstart = 20)
km.cluster <- km.out$cluster
names(km.cluster) <- ncilab
plot(km.cluster)
